# Memoria - Listas (PWA)
Prototipo de página para entrenar y desarrollar tu memoria (mental) en sus diferentes fases. Contactáctame si tienes sugerencias para mejorarla: Christian.espinosa.9511@gmail.com

MVP de una PWA para entrenamiento de memoria con listas de palabras.

## Qué hace
- Muestra una lista por tiempo limitado (memorización).
- Permite escribir lo recordado (recall activo).
- Califica aciertos, faltantes y extras.
- Ajusta dificultad por nivel y guarda progreso localmente.
